# Friday AI Assistant

Friday is a personal AI assistant built in Python and powered by Ollama.

Designed to run locally, Friday focuses on privacy, customization, and expandability while providing AI-powered conversations, memory, voice interaction, and automation capabilities.

## Features

* Local AI processing with Ollama
* Persistent memory system
* Voice input and output support
* Customizable assistant personality
* Lightweight Python architecture
* Extensible codebase for future integrations

## Tech Stack

* Python
* Ollama
* Gemma Models
* Vosk Speech Recognition
* JSON Memory Storage

## Installation

```bash
git clone https://github.com/bronzy1/Friday-AI-Assistant.git
cd Friday-AI-Assistant
pip install -r requirements.txt
```

Make sure Ollama is installed and the required model is available before running Friday.

## Usage

```bash
python friday.py
```
